# Logics Capability Guide

이 문서는 기존 custom logic flow 서사를 capability 기준으로 정리한 가이드다.

## Custom Logic via `/call`

- `:auth.sub` 시스템 변수
  - Capability: `LOGICS-001`
- public logic
  - Capability: `LOGICS-002`
- execute-only 응답 형식
  - Capability: `LOGICS-003`
- 필수 파라미터 검증
  - Capability: `LOGICS-004`
- 기본값 파라미터
  - Capability: `LOGICS-005`
- role guard
  - Capability: `LOGICS-006`
- 오류 처리
  - Capability: `LOGICS-007`
