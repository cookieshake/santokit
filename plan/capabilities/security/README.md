# Security Capability Guide

이 문서는 기존 security flow 서사를 capability 기준으로 정리한 가이드다.

## CEL Conditions

- resource 조건을 SQL 필터에 반영
- Capability: `SECURITY-001`, `SECURITY-002`, `SECURITY-003`

## Column Visibility

- role별 컬럼 노출 제한
- Capability: `SECURITY-004`

## Column-Level Permissions

- columns 기반 접근 제어
- Capability: `SECURITY-005`
